## 项目简介

本系统是针对纺织行业设备维护需求开发的预测性维护系统，通过对轴承振动信号的智能分析，实现：

- **故障诊断**：自动识别轴承故障类型（正常/内圈/外圈/滚动体）
- **寿命预测**：预测轴承剩余使用寿命（RUL）
- **状态监控**：实时监控设备健康状态
- **维护建议**：根据预测结果提供维护决策支持

核心算法：
- 故障分类：XGBoost（测试准确率 **100%**）
- 寿命预测：LSTM神经网络（R² = **0.87**）
- 跨域泛化：**DANN领域自适应（NASA R²=0.95, XJTU R²=0.99）**

## 模型性能

### 故障分类模型（XGBoost）

| 指标 | 数值 |
|:---|:---:|
| 测试集准确率 | **100%** |
| F1分数 | 1.0000 |
| 特征维度 | 72维（含包络谱） |
| 类别数 | 4类 |

### RUL预测模型对比

| 模型 | NASA R² | XJTU R² | 说明 |
|:---|:---:|:---:|:---|
| **DANN** | **0.95** | **0.99** | 推荐，跨数据集泛化 |
| LSTM (NASA) | 0.98 | -0.89 | 仅NASA数据集有效 |
| LSTM (迁移) | 0.98 | 0.16 | 迁移学习效果有限 |

DANN优势：单一模型支持多数据集，NASA和XJTU均有高精度，工况变化适应性强。

## 技术栈

- **后端**: Flask 3.0, Python
- **前端**: Vue 3 (Composition API), Vite 5.0, Axios
- **模型**: XGBoost 2.0, PyTorch 2.0 (LSTM)
- **信号处理**: SciPy, NumPy, PyWavelets
- **特征工程**: 58维（时域+频域+时频域）
- **数据可视化**: Chart.js
