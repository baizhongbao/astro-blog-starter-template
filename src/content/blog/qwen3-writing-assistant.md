## 项目简介

基于本地 Qwen3-4B 模型的中文写作工具，支持散文、诗歌、故事等多种文体。

## 文件说明

| 文件 | 用途 |
|------|------|
| `启动写作助手.bat` | 双击启动（推荐） |
| `write.py` | 主程序（单文件，包含所有逻辑） |
| `output.txt` | 生成的文章 |

## 使用方法

### 方式一：双击启动

直接双击 `启动写作助手.bat`

### 方式二：命令行

```bash
# 交互模式
python write.py

# 快速模式
python write.py 主题 字数 文体
python write.py 春天 150 诗歌
python write.py 秋天的落叶 300 散文
```

## 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| 主题 | 写作主题 | 春天、月光、大海 |
| 字数 | 文章字数 | 150、300、600 |
| 文体 | 文章类型 | 散文、诗歌、故事、对话、说明文 |

## 文体类型

1. 散文
2. 对话
3. 说明文
4. 故事
5. 诗歌

## 示例输出

**主题：月光，文体：诗歌**

> 银辉洒落夜无垠，
> 清影徘徊梦里人。
> 风起云舒皆入画，
> 月华如水润心神。

**主题：飞行，文体：散文**

> 飞机划过天际，留下一道白色的痕迹，像是在蓝色画布上勾勒出的短暂诗行。云层之上，阳光纯净得没有一丝杂质，照在机翼上泛起银色的光芒。乘客们透过舷窗俯瞰大地，城市变成了棋盘，河流变成了丝带，而那些平常的烦恼，似乎也随着高度的攀升，变得渺小而遥远。

## 技术细节

- **模型**: Qwen3-4B-Q4_K_M.gguf
- **推理**: llama-cli (CUDA 加速)
- **思维链控制**: `/no_think` 禁用 Qwen3 思维链
- **生成速度**: ~37 tokens/s
- **架构**: 单文件 Python 脚本，自动检测生成完成

## 注意事项

1. 确保已安装 Python 3.10+
2. 需要支持 CUDA 的 NVIDIA 显卡
3. 首次生成需要加载模型，约需 10-20 秒
